Scheduling Formulas

  • Turnaround = finish time - arrival time
  • Waiting = turnaround - total CPU burst
  • Response = first CPU start - arrival time
  • Throughput = completed processes / total time

Algorithm Rules

  • FCFS: first ready process runs first.
  • SJF: shortest burst among ready processes.
  • SRTF: shortest remaining time, preemptive.
  • RR: circular queue with a fixed quantum.
  • HPF: highest priority, preemptive or not.

Memory Keywords

  • MMU: converts logical address to physical address.
  • Swapping: moves processes between RAM and disk.
  • Fragmentation: wasted space, internal or external.
  • Paging: pages go into frames.
  • Virtual memory: runs programs larger than RAM.
